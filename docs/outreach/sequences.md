## Notes d'exécution

### Personnalisation obligatoire avant envoi
- [ ] Remplacer tous les `[NOM]`, `[NOM AE]`, `[LIEN CALENDLY]`
- [ ] Vérifier l'actualité des "data points" mentionnés dans les relances J3
- [ ] Adapter l'objet email si l'actualité du prospect a évolué
- [ ] Vérifier le titre exact de la persona sur LinkedIn avant envoi

### Règles de tracking HubSpot
- Créer un Contact + Company pour chaque prospect avant J0
- Associer chaque email à la Deal pipeline "Outreach Wave 1"
- Logger chaque action (email envoyé, ouvert, répondu) dans HubSpot
- Propriété `caelum_pilot_status` = `not_started` au démarrage

### Si réponse positive
→ Envoyer invitation Calendly immédiatement
→ Préparer démo personnalisée avec les engines DDH pertinents pour le secteur
→ Mettre à jour Deal stage → "Démo planifiée"
→ Notifier Customer Success pour préparer l'onboarding pilote
