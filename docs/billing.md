## Outil de facturation

À valider entre :

### Option A — Stripe Billing
- **Avantages :** intégration directe avec le site, paiement carte, gestion abonnements, webhooks
- **Inconvénients :** moins adapté facturation B2B SEPA, nécessite développement
- **Coût :** 0.5% + frais Stripe par transaction

### Option B — Pennylane
- **Avantages :** outil comptable français/belge, virement SEPA natif, factures conformes normes EU
- **Inconvénients :** moins d'automatisation, intégration manuelle possible
- **Coût :** à partir de 49€/mois

### Recommandation
Combiner les deux : **Stripe** pour les paiements carte/automatisation + **Pennylane** pour la comptabilité et les virements SEPA. À valider avec le comptable.
