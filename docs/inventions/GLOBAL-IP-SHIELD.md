## Tribunal Unifié des Brevets (UPC) — Révolution 2023

Depuis juin 2023, l'UPC change TOUT pour les inventeurs européens :

**Avant l'UPC :** Pour attaquer un contrefacteur opérant dans 5 pays UE → 5 procès séparés → coût €500k+ → 5-10 ans.

**Avec l'UPC :** 1 seul procès à Luxembourg / Munich / Paris → décision applicable dans TOUS les pays UE participants → coût €50k-100k → 12-24 mois.

**Pour Caelum :** Si SAP, Oracle ou une startup copie nos engines dans plusieurs pays UE → 1 seule procédure, dommages cumulés de tous les pays.
