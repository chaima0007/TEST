# Stratégie Forêt de Brevets — Caelum Partners SPRL

**Inventrice :** Chaima Mhadbi  
**Titulaire :** Caelum Partners SPRL  
**Objectif :** 10 ans d'avance permanente via dépôts tous les 6 mois

## Principe — La Forêt de Brevets

Au lieu d'un seul brevet isolé (facile à contourner), Caelum construit une **forêt de brevets interdépendants** où chaque invention rend les précédentes plus fortes et les contournements impossibles.

```
G1 (Jan 2025) ──→ G2 (Jul 2025) ──→ G3 (Jan 2026) ──→ G4 (Jul 2026)
     ↓                   ↓                  ↓                  ↓
  Base IA DH      + Fédéré/Chain      + ESG/Conflit      + GNN/Prédictif
  
G5 (Jan 2027) ──→ G6 (Jul 2027) ──→ G7 (Jan 2028) ──→ G8 (Jul 2028)
     ↓                   ↓                  ↓                  ↓
 + Génératif IA    + Zero-Knowledge    + Quantum-Safe      + Traçabilité

G9 (Jan 2029) ──→ G10 (Jul 2029) ──→ G11... (continu)
```

## Règle des 6 Mois

Tous les 6 mois, Caelum dépose :
1. Une **continuation** de l'invention parent (protège les améliorations)
2. Une **CIP** (Continuation-in-Part) avec de nouvelles revendications
3. Une **publication défensive** pour les concepts non-brevetables

Résultat : la protection ne s'arrête jamais. Quand G1 expire en 2045, G10 est encore protégé jusqu'en 2049.

## Modèle Économique — Licensing Obligatoire

### Principe
Toute entreprise qui utilise une technologie couverte par nos brevets **doit obtenir une licence**. Pas de licence = violation = procédure judiciaire.

### Grille Tarifaire

| Segment | Tarif/an | Exclusivité |
|---------|----------|-------------|
| Startup / ONG | €10k - €30k | Non |
| PME ESG / Conseil | €50k - €150k | Non |
| Grande entreprise | €200k - €500k | Non |
| Banque / Assurance | €300k - €800k | Non |
| Gouvernement / OI | €500k - €2M | Contrat-cadre |
| Exclusivité sectorielle | €1M - €5M | Oui (1 secteur) |

### Exemple concret — CAE-INV-005 (CSDDD Platform)
- SAP veut implémenter le scoring CSDDD → **€400k/an** (non-exclusif)
- BNP Paribas veut l'exclusivité bancaire → **€3M/an**
- Si SAP utilise sans licence → Tribunal de commerce Bruxelles + ITC USA

## Ce que les Concurrents Ne Peuvent PAS Faire

1. **SAP/Oracle copient CAE-INV-001** → Violation G06N 20/00 → procédure ITC bloque import USA
2. **Palantir améliore notre algo** → CIP G2/G3 couvre les améliorations → toujours notre licence
3. **Startup fait un workaround** → G4/G5 ferme les gaps → workaround devient aussi protégé
4. **Gouvernement veut utiliser sans payer** → Licences souveraines possibles mais €500k min

**Résultat : La forêt grossit plus vite que les concurrents ne peuvent contourner.**
