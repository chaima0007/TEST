# Persona : Compliance ONG / Responsable Conformité Bailleur

## Profil type
- **Titre** : Compliance Officer / Grants Manager / Responsable Conformité & Redevabilité
- **Formation** : Sciences humaines, droit, développement international, Sciences Po / Sorbonne
- **Ancienneté poste** : 2-7 ans
- **Contexte** : ONG humanitaire ou droits humains, budget 5-50M€/an, financements UE/USAID/AFD
- **Reporting** : Directeur Général ou Directeur des Opérations

## Motivations profondes
1. **Impact réel et mesurable** : pas de reporting vide — preuves sur le terrain
2. **Conformité bailleurs** : respecter exactement les exigences de chaque bailleur (UE, USAID, AFD)
3. **Budget serré** : maximiser l'impact par euro dépensé (overhead <15% pour la plupart)
4. **Redevabilité populations** : droits des bénéficiaires prime sur tout autre objectif
5. **Protéger l'organisation** : scandal = perte financement = fermeture

## Douleurs prioritaires
- Budget compliance minimal vs exigences croissantes bailleurs
- Systèmes de monitoring sur le terrain = Excel, WhatsApp, pas d'outil adapté
- Reporting bailleurs : formats hétérogènes, données identiques raisies 3 fois
- RGPD appliqué aux bénéficiaires vulnérables (réfugiés, survivants VBG) : tension protection
- Turnover élevé = perte de mémoire institutionnelle

## Ce qui rassure le Responsable Compliance ONG
- Prix adapté (modèle ONG : <50% tarif corporate pour même usage)
- Formation incluse et accompagnement terrain
- Interopérabilité avec outils ONG standards (KoboToolbox, DHIS2, CommCare)
- Données hébergées RGPD, consentement bénéficiaires intégré
- Références autres ONG (Médecins Sans Frontières, ACF, IRC)

## Ce qui irrite le Responsable Compliance ONG
- Tarifs corporate sans adaptation ONG
- Solutions orientées corporate ESG sans compréhension contexte humanitaire
- Promesses d'impact sans métriques concrètes terrain
- Processus d'achat longs (pas de budget pour un cycle de vente de 12 mois)
- Ignorer les contraintes opérationnelles terrain (connexion faible, terrain hostile)

## Argument d'ouverture recommandé
> "Nous proposons un tarif ONG à 40% du prix corporate, avec formation terrain incluse. Médecins du Monde utilise notre module monitoring droits humains pour leur rapport AFD — cela peut correspondre à votre cycle de reporting bailleur ?"

## Canal préféré
- Email direct et concis
- Référence par pairs ONG = extrêmement efficace
- Webinaire gratuit ouvert aux ONG
- Conférences sectorielles (ECHO Forum, IASC, Coordination SUD)

## Phrase à éviter absolument
- "Notre solution est utilisée par les plus grands groupes CAC40." (pas pertinent)
- "Le ROI est de X% sur 3 ans." (cycle budgétaire ONG = 1 an, RH non calculable)

## Rôle dans le cycle de vente
- Décideur pour achats <30K€ (Grants Manager souvent autonome)
- Co-décideur avec DG pour achats >30K€
- Budget bailleur = earmarked, opportunité en cours d'année si ligne "compliance tools" existe
- Accélérateur : montrer que l'outil répond à une exigence bailleur précise = vente rapide
