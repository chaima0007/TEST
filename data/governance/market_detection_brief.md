# 📡 Détection du marché — signaux & opportunités

*Pas de surveillance autonome 24/7 : la détection est exécutée par l'assistant via recherche web à chaque passage. Cadence recommandée : mensuelle. Dernière détection : 2026-06-26.*

## Synthèse : 4 signaux (1 en priorité haute)

### 🔴 HAUTE — réglementation/lois
- **Signal** : Wallonie — Réforme PEB : dès le 1er avril 2026, les logements classés G ne peuvent plus être (re)loués lors d'un 'nouveau bail' (y compris reconduction, changement de locataire, relocation après vacance).
- **Impact** : Confusion massive des propriétaires sur ce qui compte comme 'nouveau bail' → forte demande d'info claire.
- **Opportunité** : Créer un thème 'PEB / performance énergétique & location' sourcé + un mini-diagnostic 'puis-je encore louer ?'.
- Source : https://notrebelgique.be/je-pensais-etre-en-regle-cette-nouvelle-reforme-qui-change-tout-pour-les-proprietaires-belges-des-avril-2026/ (vu le 2026-06-26)

### 🟠 MOYENNE — réglementation/lois
- **Signal** : Bruxelles — Nouvelles normes minimales de qualité (sonnette/boîte aux lettres, 2 points d'eau chaude+froide, conformité gaz, chauffage & eau à charge du bailleur).
- **Impact** : Renforcement de la protection locataire → besoin d'info à jour.
- **Opportunité** : Enrichir le thème 'salubrité' Bruxelles avec ces critères précis.
- Source : https://be.brussels/en/housing/amendment-leasing-legislation-november-2024 (vu le 2026-06-26)

### 🟠 MOYENNE — réglementation/lois
- **Signal** : Énergie — Interdiction d'installer des chaudières mazout/charbon dans le neuf dès le 1er janvier 2026.
- **Impact** : Transition énergétique impactant logement.
- **Opportunité** : Lien avec le thème PEB/énergie.
- Source : https://weinvest.be/fr-BE/blog/b2c/real-estate-changes (vu le 2026-06-26)
